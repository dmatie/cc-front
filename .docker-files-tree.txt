🚀 DÉMARRAGE RAPIDE

Option 1 (Script automatique):
  $ ./docker-build.sh

Option 2 (Docker Compose):
  $ docker-compose up -d

Option 3 (Commandes manuelles):
  $ docker build -t ccapp:latest .
  $ docker run -d -p 8080:80 --name ccapp-container ccapp:latest

Accès: http://localhost:8080
